HTML Basic

In this section, we cover basic tags and the anatomy of an HTML document.

Basic Tags

  • <html>: Root element of an HTML document.
  • <head>: Contains meta information about the document.
  • <body>: Contains the content of the page.

Attributes and Entities

Attributes provide additional information about an element, while entities are used to display reserved characters or symbols.
